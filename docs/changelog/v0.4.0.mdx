This release makes provider operation identifiers read naturally in TypeScript while retaining Alchemy's shared-generator, pinned-spec, committed-output architecture.

## Packages

- [`@kevinmichaelchen/distilled@0.1.3`](https://www.npmjs.com/package/@kevinmichaelchen/distilled) ΓÇË normalized operation naming plus stale-output cleanup
- [`@kevinmichaelchen/distilled-avalara@0.1.2`](https://www.npmjs.com/package/@kevinmichaelchen/distilled-avalara) ΓÇË lower camelCase AvaTax operations
- [`@kevinmichaelchen/distilled-github@0.1.2`](https://www.npmjs.com/package/@kevinmichaelchen/distilled-github) ΓÇË normalized slash-delimited REST operation IDs
- [`@kevinmichaelchen/distilled-jira@0.1.2`](https://www.npmjs.com/package/@kevinmichaelchen/distilled-jira) ΓÇË normalized dotted Atlassian operation IDs
- [`@kevinmichaelchen/distilled-opensearch@0.1.2`](https://www.npmjs.com/package/@kevinmichaelchen/distilled-opensearch) ΓÇË namespace-aware OpenSearch operation casing
- [`@kevinmichaelchen/distilled-statsig@0.1.2`](https://www.npmjs.com/package/@kevinmichaelchen/distilled-statsig) ΓÇË lower camelCase names derived from tags and summaries

## Changed

- Dots, slashes, hyphens, underscores, and whitespace now form deterministic camelCase word boundaries.
- Operation functions begin with a lowercase character; generated schema and type names remain PascalCase.
- Successful generation atomically replaces the complete resource-service tree, so removed or case-renamed operations cannot survive as stale modules.
- Avalara, GitHub REST, Jira, OpenSearch, and Statsig were regenerated. Auth0 and Slack already produced idiomatic names and did not require package releases.

This is intentionally a breaking cleanup with no deprecated aliases. The packages have no known consumers, so carrying the malformed names forward would add complexity without protecting real users.

## Examples

- AvaTax: `TaxRatesByAddress` Γ¶Ê `taxRatesByAddress`
- GitHub: `reposget` Γ¶Ê `reposGet`
- Statsig: `GatesListGates` Γ¶Ê `gatesListGates`
- OpenSearch: `clusterhealth0` Γ¶Ê `clusterHealth0`
- Jira: `AddonPropertiesResource.getAddonProperties_get` Γ¶Ê `addonPropertiesResourceGetAddonPropertiesGet`

Official numeric OpenSearch route suffixes remain. They distinguish separate path and HTTP-method variants in the source contract; the generator does not invent overloads or merge operations.

## Verified

Every affected SDK regenerates from its pinned source, typechecks, passes provider tests, builds its publishable output, and is released independently through GitHub OIDC trusted publishing.
