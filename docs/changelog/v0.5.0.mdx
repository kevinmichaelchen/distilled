This release adds Basis Theory while preserving the separate-source, pinned-submodule, committed-output, and independently published architecture learned from Alchemy.

## Package

- [`@kevinmichaelchen/distilled-basis-theory@0.1.0`](https://www.npmjs.com/package/@kevinmichaelchen/distilled-basis-theory) — 117 Effect-native Basis Theory API operations

## Added

- An immutable private mirror of Basis Theory's official production-served OpenAPI 3.0.1 document
- Operations for token lifecycles, token intents, applications, proxies, reactors, sessions, 3DS, network tokens, wallets, tenants, and logs
- Production and test API addressing with an explicit custom-base-URL override
- Redacted `BASIS_THEORY_API_KEY` credentials injected through the official `BT-API-KEY` header
- Problem-details error decoding, shared typed HTTP failures, retry hints, and provider retry services
- A dedicated SDK guide and service announcement with three common token-vault workflows

Two upstream-deprecated operations are intentionally excluded through the same `skipDeprecated` generator policy used by the rest of the factory.
