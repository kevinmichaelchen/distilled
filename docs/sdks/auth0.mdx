The Auth0 SDK pins the official Management API v2 contract and keeps tenant addressing and bearer-token policy in the handwritten provider layer.

<CodeGroup>

```bash npm
npm install @kevinmichaelchen/distilled-auth0 effect
```

```bash pnpm
pnpm add @kevinmichaelchen/distilled-auth0 effect
```

```bash bun
bun add @kevinmichaelchen/distilled-auth0 effect
```

</CodeGroup>

## Authentication

Set `AUTH0_DOMAIN` and `AUTH0_MANAGEMENT_TOKEN`. The client targets `https://<tenant>/api/v2` and sends the token as bearer authentication.
