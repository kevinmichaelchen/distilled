The Avalara SDK pins the official AvaTax REST v2 Swagger document in a dedicated source repository, then adds only the handwritten authentication, errors, and retry behavior that belong to the provider boundary.

<CodeGroup>

```bash npm
npm install @kevinmichaelchen/distilled-avalara effect
```

```bash pnpm
pnpm add @kevinmichaelchen/distilled-avalara effect
```

```bash bun
bun add @kevinmichaelchen/distilled-avalara effect
```

</CodeGroup>

## Authentication

Use either `AVALARA_BEARER_TOKEN` or the AvaTax account pair `AVALARA_ACCOUNT_ID` and `AVALARA_LICENSE_KEY`. `AVALARA_BASE_URL` can select a sandbox or another AvaTax endpoint.
