<Badge variant="success">Published</Badge> <Badge>OpenAPI 3</Badge>

GitHub maintains its REST API description in a large upstream repository. The spec mirror contains only the exact versioned bundle consumed by generation, avoiding an impractical downstream submodule while preserving authority.

Slash-delimited operation IDs become lower camelCase exports such as `reposGet`, `issuesListForRepo`, and `pullsCreate`.

<CodeGroup>

```bash npm
npm install @kevinmichaelchen/distilled-github effect
```

```bash pnpm
pnpm add @kevinmichaelchen/distilled-github effect
```

```bash bun
bun add @kevinmichaelchen/distilled-github effect
```

</CodeGroup>
