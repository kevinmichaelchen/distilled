## Shared qualities

<Columns cols={2}>
  <Column>
    <Panel title="Effect-native">
      Runtime schemas, typed errors, categorized retries, tracing, and yieldable operations are first-class rather than wrapped after generation.
    </Panel>
  </Column>
  <Column>
    <Panel title="Source-reviewable">
      Each package pins an exact upstream source and commits its generated output for review.
    </Panel>
  </Column>
</Columns>

All packages expose TypeScript source to Bun and compiled ESM plus declarations to standard Node runtimes.

[Compare versions, authentication, and source strategies](/sdks/support-matrix).

:::note[Repository access]
The npm packages are public and include generated TypeScript source. The sibling implementation and mirrored-specification repositories are currently private; links to them are most useful to maintainers with access.
:::
