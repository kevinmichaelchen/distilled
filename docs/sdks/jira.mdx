The Jira SDK mirrors Atlassian's authoritative OpenAPI document in a dedicated spec repository, pins that mirror as a submodule, applies documented patches, and generates Effect operations through the shared factory.

<CodeGroup>

```bash npm
npm install @kevinmichaelchen/distilled-jira effect
```

```bash pnpm
pnpm add @kevinmichaelchen/distilled-jira effect
```

```bash bun
bun add @kevinmichaelchen/distilled-jira effect
```

</CodeGroup>

## Why a spec repository?

Atlassian publishes a maintained OpenAPI artifact that is compact enough to mirror directly. Isolating it gives the SDK a stable, exact source pin and makes upstream refreshes explicit.
