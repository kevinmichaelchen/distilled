Slack does not provide a maintained OpenAPI document suitable for this factory. Instead, the implementation pins `slackapi/node-slack-sdk` directly and derives its Web API surface with a provider-specific TypeScript AST generator.

<CodeGroup>

```bash npm
npm install @kevinmichaelchen/distilled-slack effect
```

```bash pnpm
pnpm add @kevinmichaelchen/distilled-slack effect
```

```bash bun
bun add @kevinmichaelchen/distilled-slack effect
```

</CodeGroup>

## Why there is no distilled-spec-slack

The official source repository is already the compact, authoritative typed input. A second mirror would add indirection without adding curation, stability, or reviewability.
